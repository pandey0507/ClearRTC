# ClearRTC 🔴

A Zoom-like real-time video communication app built with MERN Stack.

## 💻 Tech Stack
- *Frontend*: React.js
- *Backend*: Node.js + Express
- *Database*: MongoDB Atlas
- *WebRTC*: For real-time video calling

## 🚀 Features
- User Signup/Login
- Join Meeting as Guest
- Video & Audio Calling
- Real-time Meeting Rooms

## 🌐 Deployment
- Frontend: Vercel
- Backend: Render
